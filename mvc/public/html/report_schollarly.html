1.Abstract


Scholarly HTML is a domain-specific rich document format built entirely on
open standards that enables the interoperable exchange of scholarly articles in a manner that is compatible with off-the-shelf browsers..


2.Cerinta proiect


Sa se dezvolte o aplicatie Web pentru gestiunea vizitelor de care beneficiaza persoanel condamnate la executarea unei pedepse intr-un penitenciar sau intr-o casa de corectie
Fiecarei vizite i sevor atasa informatii privind individul care efectueaza vizita --minimal se vor considera:identitatea,fotografia,relatia cu condamnatul (ruda,tutore,avocat,prieten),natura vizitei.
De asemenea,vor fi consemnate:data realizarii vizitei,natura si durata acesteia, posibile obiecte/date furnizate condamnatului sau oferite de condamnat vizitatorului, rezumatul discutiilor efectuate. Instrumentul implementat va genera statistici--documente HTML,CSV,JSON-- vizand vizitele realizate per individ,perioada de timp.


Limbaje folosite

  • HTML5 utilizat pentru structurarea continutului
  • CSS/CSS3 utilizat pentru stilizarea paginilor
  • JavaScript, Ajax
  • PHP utilizat pentru back-end, pentru comunicarea cu serverul Apache si baza de date MySQL
  • MySQL utilizat pentru managementul bazei de date

3.Dezvoltare proiect

3.1.Baza de date

Tabele baza de date:

  • Users: stocarea userilor
  • Visits: stocarea vizitelor
  • Detained: stocarea detinutilor din penitenciar
  • Comments: stocarea comentariilor lasate de useri

Am stabilit structura proiectului si aspectele necesare pentru funcionarea proiectului Am creat o baza de date de tip MySQL cu diferite tabele(users,visits,detained,comments) Am realizat un model de aplicatie actori,roluri si diagrame uml pentru a identifica mai usor fiecare componenta a proiectului.
Am inceput cu realizarea paginilor pentru user,si stilizarea acestora. Aplicatia urma sa fie dezvolatata dupa acel model.

3.2.Functionalitati aplicatie

  • Register utilizatorii se pot inregistra si loga in aplicatie pentru a putea crea o noua vizita,altfel poate vizualiza numai homepage.
  • Login utilizatorii se pot loga in aplicatie pentru a putea crea o noua vizita,pentru a vizualiza visitele sale.
  • New visit utilizatorii pot crea o noua vizita
  • My visits utilizatorii pot vizualiza lista cu vizitele lor,pot sterge o anumita vizita,si downloada in format csv,lista vizitelor lor.
  • Statistics Admin pot vizualiza diverse statistici despre vizite,pe categorii:
    • numar de vizite de familie
    • numar de vizite de plan legal
    • numar de vizite de prietenie
    • numar de vizite facute de avocat
    • numar de vizite facute de rude
    • numar de vizite facute de prieteni
  • Contact utilizatorii ne pot contacta prin email.
  • Homepage
    • Userii pot genera un citat random prin apasare de buton,fara a da reload pagini_inainte_logare
    • Userii pot vizualiza o galerie e imagini si downloada imaginile
    • se pot vizualiza cele mai noi informatii despre cele mai noi vloguri de pe youtube
  • Modul de administrare
    • Administratorul poate vizualiza toti userii,vizitele si detinutii
    • Administratorul poate sterge orice cont de user,odata cu stergerea unui cont se vor sterge si vizitele acestuia
    • Administratorul poate downloada liste in format csv cu toate vizitele,toti userii si toti detinutii
    • Administratorul poate downloada statistici despre visite
    • Administratorul poate vizualiza un clasament top 3 cele mai lungi condamnari
    • Administratorul poate downloada lista in format csv cu toate vizitele unui anumit detinut
